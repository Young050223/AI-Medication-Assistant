# 热更新开发模式

启动 Capacitor iOS 热更新开发模式，修改代码后自动刷新模拟器。

## 步骤

// turbo-all

1. 首先启动 Vite 开发服务器（在后台运行）：
```bash
cd /Users/mac/Desktop/AI-Medication-Assistant && npm run dev
```

2. 等待开发服务器启动后，启动 iOS 模拟器并连接热更新：
```bash
cd /Users/mac/Desktop/AI-Medication-Assistant && npx cap run ios --livereload --external
```

## 说明

- 开发服务器会在 `http://localhost:5173` 运行
- iOS 模拟器会自动连接到开发服务器
- 修改 `src/` 目录下的文件后，模拟器会自动刷新
- 按 `Ctrl+C` 停止开发服务器

## 注意事项

- 确保模拟器和电脑在同一网络
- 如果热更新失效，尝试重启开发服务器
- 首次运行可能需要在 Xcode 中信任开发者证书
